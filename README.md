# SistemaInventarioMVC

Sistema de inventario web desarrollado con Java, JSP, Servlets, MySQL y arquitectura MVC.

## Funcionalidades
- Login de usuarios
- Registro de productos
- Edición y eliminación de productos
- Búsqueda por nombre/categoría
- Paginación de productos
- Diseño responsive profesional

## Tecnologías utilizadas
- Java Web
- JSP
- Servlets
- MySQL
- Apache Tomcat
- HTML/CSS
- Maven
